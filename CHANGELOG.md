## Project History

PhoenixBoot (also known as PhoenixGuard) is a production-ready firmware defense system designed to protect against bootkits, rootkits, and supply chain attacks.

### Key Features
- Hardware-level firmware recovery
- Secure boot enforcement
- Complete UEFI boot chain with cryptographic verification
- Container-based architecture
- Terminal User Interface (TUI)
- SecureBoot bootable media creator
