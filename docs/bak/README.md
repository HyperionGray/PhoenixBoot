# Archived documentation

This directory holds older or superseded documentation that is still useful for historical reference but should not be treated as the current source of truth for the OSS release.

Use the actively maintained documentation instead:

- `../README.md` - primary documentation index inside `docs/`
- `../CHANGELOG.md` - current project changelog
- `../FEATURES.md` - current feature status
- `../reviews/` - organized review and audit documents
- `../implementation/` - implementation notes and testing summaries

Archived files are kept here when they reference older paths, older concepts, or planning material that still has reference value.
